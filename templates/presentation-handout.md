# Шаблон: презентация-раздатка (self-sufficient handout)

Выработан 2026-07-20 на презентации для ДИТ по эталону ROOMY (raw-пример у Д. Смолева).
Применять для всех презентаций, которые остаются у заказчика «на столе».

## Главный принцип

Слайд должен читаться без докладчика. Никаких «оторванных слов» — каждый элемент
это законченное предложение с подлежащим и сказуемым.

## Анатомия слайда

1. **Кикер** (вверху): `★ РУБРИКА` — маленькими прописными, красная звёздочка.
   Рубрики образуют оглавление: КОНТЕКСТ → ПРОБЛЕМА → ПРЕДЛОЖЕНИЕ → КАК УСТРОЕНО → …
2. **Заголовок-утверждение** (28–32pt, жирный): тезис, а не тема. Не «Протоколы»,
   а «MCP отвечает за данные, A2A — за координацию».
3. **Подзаголовок** (13pt, серый): 1–2 полных предложения, раскрывающих тезис.
4. **Контентная зона** — варианты, чередовать:
   - тёмная карточка-акцент слева + нумерованный список 01/02/03 справа
     (красный номер, жирный заголовок пункта, описание 1–2 предложениями, тонкий разделитель);
   - сравнение двух карточек (светлая «до» → стрелка → тёмная «после»);
   - горизонтальный таймлайн с нумерованными кругами и подписями-предложениями;
   - таблица сравнения с зелёной колонкой «наша» позиция.
5. **Вывод-плашка** внизу (обязательно!): одно жирное предложение — что унести
   с этого слайда. Тёмно-зелёная с белым или светло-мятная с зелёным.
6. **Футер**: имя компании слева (засечковый жирный), номер слайда справа, тонкая линия сверху.

## Палитра

Глубокий зелёный `14584A` (доминанта), тёмнее `0F473B`, мята `E7F2EE`, светло-серый `F3F6F5`,
красный акцент `C63D2F` (только номера и звёздочки), текст `1E2725`, серый `5E6D68`.
Тёмные слайды — первый и последний («сэндвич»).

## Антипаттерны (из ревью v1 деки для ДИТ)

- Слова-обрывки вместо предложений («Заявка», «описание задачи») — недопустимо.
- Копирование чужого стиля без осмысления.
- Иконки ради иконок; нумерация 01/02/03 работает лучше.
- Слайд без вывода-плашки — слайд без смысла для читателя раздатки.
