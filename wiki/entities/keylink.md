# Keylink

Второй прикладной продукт на [[oxtech-platform]] (после [[oxdocs]]): веб-сервис автоматизации
закупок электронных компонентов для компании «Кейлинк» (smart-components.pro).
Go 1.26 + React 19 + Vite.

Значение для позиционирования: подтверждает тезис «одна платформа — разные прикладные
продукты». [[oxdocs]] — документооборот, Keylink — отраслевой процесс закупок; обе вертикали
используют один слой [[mcp]], одних агентов и один [[oxmodel]].

## Функциональность

- Process Viewer: бизнес-процессы, подзадачи, чек-поинты, точки [[hitl]].
- Встроенный почтовый клиент (robot@smart-components.pro).
- Справочники: поставщики, компоненты, даташиты.
- Формирование коммерческих предложений с наценкой, предпросмотр PDF.
- Procurement Agent (A2A) для автоматизации workflow — build tag `procurement`.

## Интеграция с платформой

Данные — через отдельный MCP-сервер `oxteam.me/mcp/keylink`: **75 инструментов**, 13 concern-модулей,
крупнейшие домены — email (17), auth (10), компоненты (8), цены (7). Схема БД `keylink`.

⚠️ Keylink MCP имеет собственную аутентификацию (`keylink_bearer_authentication.rb`) и **не
покрыт механизмом `AgentToolPermission`** — белый список инструментов на агента, работающий в
Portal MCP, на этот контур не распространяется. См. [[mcp]].

## Статус
