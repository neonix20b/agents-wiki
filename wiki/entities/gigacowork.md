# GigaCowork (Сбер)

Платформа управления ИИ-агентами «для всей компании» от «Салют для бизнеса» (экосистема Сбера). Конкурент [[oxtech-platform]]. Запуск в открытый тестовый доступ — 19.05.2026 на ЦИПР. Пресса называет её «первым прямым российским ответом Claude Cowork».

## Продукт
- **No-code**: агенты настраиваются через веб-интерфейс «за минуты»; логика задаётся обычным языком в чате как регламент/инструкция/чек-лист (данные, правила, точки возврата на проверку, требуемый результат).
- Описания становятся **«навыками»**, которые переиспользуют другие сотрудники и подразделения.
- **Интеграции по MCP**: коннекторы к CRM, ERP, почте, файловым хранилищам; анонсированы 1С, СБИС, Битрикс, банковские API РФ — см. [[mcp]].
- **Human-in-the-loop**: агент — помощник, финальное решение за сотрудником.
- Сценарии: аналитические отчёты из корпоративных данных, расчёт проектов и рисков, подготовка документов, HR/подбор.

## Заявленные результаты пилотов
Документы +80%, экономия времени сотрудника до 81,5%, HR 83%, отчётность +70%, поиск кандидатов +93%, анализ информации −30–50% времени.

⚠️ Это внутренние пилоты Сбера, подготовленные к запуску, — не независимая выборка (критика vc.ru). Внешних кейсов на 2026-07-20 не опубликовано.

## Открытые вопросы (для мониторинга)
Глубина no-code интерфейса в реальной нагрузке; методология (не опубликована); модель развёртывания (on-premise?); ценообразование; скорость итераций.
